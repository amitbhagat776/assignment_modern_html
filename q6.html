HTML HTML5
HTML stands for Hypertext Markup Language. HTML5 stands for Hypertext Markup Language 5.
HTML4, the predecessor of HTML5, had a limited set of elements and attributes. HTML5 introduced new semantic elements like <header>, <nav>, and <section> for better document structure.
HTML4 required plugins like Flash for multimedia content. HTML5 has built-in support for audio and video elements, reducing the need for plugins.
HTML used <font> and inline styles for text formatting. HTML5 encourages the use of CSS for styling and formatting, promoting separation of content and presentation.
